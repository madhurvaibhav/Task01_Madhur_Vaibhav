# Number Guessing Game

This is a simple Java console-based Number Guessing Game developed as a beginner project.

The program generates a random number between 1 and 100, and the user has to guess the correct number. After each guess, it provides a hint whether the guessed number is too high or too low until the correct number is guessed or the maximum number of attempts is reached.

## Features

- Random number generation
- User input using Scanner
- Hint messages (Too High / Too Low)
- Limited number of attempts
- Input validation using exception handling

## Technology Used

- Java
